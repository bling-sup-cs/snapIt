SnapIt

Guia Rápido e Completo de Uso

Produtividade Precisão Segurança

O SnapIt é uma extensão desenvolvida para aumentar a produtividade na escrita, consolidando três pilares essenciais para o seu fluxo de trabalho:

Autocomplete

(Expansão de Texto)

Spellcheck

(Corretor Ortográfico)

Autosave

(Recuperação Automática)

1. Funcionalidades Principais

1.1. Autocomplete (Expansão de Texto)

Crie Snaps (comandos curtos) que se expandem para textos longos predefinidos. Ideal para saudações e respostas padrão.

Exemplo Prático:
Digitando /saudacao o sistema insere automaticamente o texto configurado.

1.2. Spellcheck (Corretor Ortográfico)

Corretor integrado focado no domínio do Bling para garantir profissionalismo total.

  • Ativação: Clique no ícone de ativação no header da extensão (necessário apenas uma vez).
  • Inteligência: Sugere correções em tempo real e evita erros comuns.
Navegação por Teclado:
  • OPÇÃO Abre o menu de sugestões (tecla próxima ao Alt/Setas).
  • ↑ ↓ Navega entre as sugestões disponíveis.
  • ENTER Seleciona e aplica a correção sugerida.
  • ESC Fecha o menu de sugestões sem aplicar alterações.

1.3. Autosave (Recuperação de Texto)

Nunca mais perca textos por fechamento acidental de abas ou quedas de energia.

Recuperação Imediata: Use o atalho CTRL + Z para restaurar o último conteúdo digitado.

1.4. Formatador de Textos

Utilize atalhos rápidos para aplicar formatação Markdown diretamente no editor de texto.

Estilo / Função Resultado Markdown Atalho de Teclado
Negrito (Bold) **texto** Ctrl + B
Itálico (Italic) *texto* Ctrl + I
Negrito e Itálico ***texto*** Ctrl + Shift + B
Código (Inline) `texto` Ctrl / Cmd + `
Bloco de Código ```texto``` Ctrl + Shift + M
Link Externo [texto](url) Ctrl + K
Inserir Imagem ![alt](url) Ctrl + Shift + G
Link via Clipboard [alt](url_do_clipboard) Ctrl + Shift + K
Ícone (Alias) Aceita alias de ícone Ctrl + Shift + I

  • Criar novos Snaps: Adicione comandos personalizados rapidamente.
  • Editar e excluir: Mantenha sua biblioteca sempre atualizada modificando ou removendo Snaps existentes.
  • Arraste e Solte: Reorganize Snaps e Pastas livremente.
  • Métricas: Visualize estatísticas de uso através do ícone .

2.2. Pesquisa Inteligente

Localize qualquer comando pesquisando pelo Título ou pelo próprio Comando.

3. Ferramentas Avançadas (⚙️)

3.1. Importação e Exportação

Gerencie sua base de Snaps de forma externa para backup, compartilhamento ou edições em massa.

📤 Exportar Snaps
  • Filtros: Por pasta ou tipo de armazenamento (Sync/Local).
  • Formatos: JSON ou CSV.
  • Segurança CSV: Formato ajustado para Excel, evitando fórmulas.
📥 Importar Snaps
  • Opções: Criar novos ou atualizar existentes.
  • Formatos: JSON ou CSV suportados.
  • Recuperação: Botão Restaurar para rollback de segurança.
Aviso para usuários Linux:
No Linux, o popup pode fechar automaticamente ao selecionar um arquivo. Caso isso ocorra, clique no ícone de monitor (Expandir para Página Inteira) no topo do popup. Na versão de página cheia, a importação funcionará normalmente.
O que o arquivo tem? O que o SnapIt faz?
ID que já existe na sua base O Snap é atualizado. Se houver campos vazios no arquivo, os valores atuais do Snap são mantidos.
ID que não existe na base O Snap é criado como novo (se a opção "Cadastrar faltantes" estiver ativa). Um novo ID será gerado.
Sem ID no arquivo O item é criado como um novo Snap.
Comando (Snap) duplicado O comando repetido é ignorado para evitar conflitos.
Limites e Requisitos de Segurança por Importação:
• Máximo de 2000 Snaps ou 25 MB por arquivo.
• Campos obrigatórios para novos Snaps: Título, Comando (Snap) e Texto.
• Limites individuais: Título (200), Comando (120) e Texto (50.000 caracteres).
• Capacidade total máxima combinada: 20.000.000 de caracteres.
