3. Migração de Dados

3.1. Exportando do Text Blaze

  • Abra o popup da extensão;
  • Clique com o botão direito → Inspecionar;
  • Acesse a aba Console;
  • No topo do console, altere de popup/ para Textblaze: Templates and Snippets;
  • Cole o script abaixo e pressione Enter.
Script Text Blaze 
chrome.storage.local.get("LOADED_DATA_CACHED", ({ LOADED_DATA_CACHED }) => {
	try {
		if (!LOADED_DATA_CACHED) { console.log("LOADED_DATA_CACHED não encontrado."); return; }
		const groups = LOADED_DATA_CACHED.groups || {};
		const snippetsByGroup = LOADED_DATA_CACHED.snippets || {};
		const readVarint = (bytes, startIndex) => {
			let result = 0; let shift = 0; let index = startIndex;
			while (index < bytes.length) {
				const byte = bytes[index]; result |= (byte & 0x7f) << shift; index++;
				if ((byte & 0x80) === 0) break; shift += 7;
			}
			return { value: result, nextIndex: index };
		};
		const cleanDecodedText = (text = "") => text.replace(/\uFFFD/g, "").replace(/[\x00-\x08\x0B-\x1F\x7F]/g, "").trim();
		const decodeDeltaText = (delta) => {
			try {
				const bytes = Uint8Array.from(delta); let index = 0;
				if (bytes[index] === 10) {
					index += 1; const first = readVarint(bytes, index); index = first.nextIndex + first.value;
					if (bytes[index] === 18) {
						index += 1; const textLen = readVarint(bytes, index); index = textLen.nextIndex;
						return cleanDecodedText(new TextDecoder("utf-8").decode(bytes.slice(index, index + textLen.value)));
					}
				}
				return cleanDecodedText(new TextDecoder("utf-8").decode(bytes));
			} catch { return ""; }
		};
		const result = [];
		Object.entries(snippetsByGroup).forEach(([groupId, snippetList]) => {
			const folderName = groups[groupId]?.name || "Sem pasta";
			snippetList.forEach((entry) => {
				const data = entry?.data || {};
				result.push({ title: data.name ?? "", command: data.shortcut ?? "", text: decodeDeltaText(data.delta), folder: folderName });
			});
		});
		const blob = new Blob([JSON.stringify(result, null, 2)], { type: "application/json" });
		const a = document.createElement("a"); a.href = URL.createObjectURL(blob); a.download = "textblaze-snippets.json"; a.click();
	} catch (error) { console.error(error); }
});

3.2. Exportando do Magical

  • Abra o popup da extensão;
  • Clique com o botão direito → Inspecionar;
  • Acesse a aba Console;
  • No topo do console, altere de popup/ para Textblaze: Templates and Snippets;
  • Cole o script abaixo e pressione Enter.
Script Magical 
chrome.storage.local.get("snippets", ({ snippets }) => {
	try {
		if (!snippets) { console.log("Nenhum snippet encontrado."); return; }
		const cleanTitle = (trigger = "") => trigger.replace(/[^a-zA-Z0-9\s]/g, "").trim();
		const result = Object.values(snippets).map((item, index) => ({
			title: cleanTitle(item.trigger),
			command: item.trigger ?? "",
			text: item.plainTextBody ?? "",
			folder: Math.floor(index / 50) === 0 ? "Favoritos" : `Favoritos ${Math.floor(index / 50) + 1}`,
		}));
		const blob = new Blob([JSON.stringify(result, null, 2)], { type: "application/json" });
		const a = document.createElement("a"); a.href = URL.createObjectURL(blob); a.download = "magical-snippets.json"; a.click();
	} catch (error) { console.error(error); }
});

4. Importação e Exportação

Gerencie sua base de Snaps de forma externa para backup, compartilhamento ou edições em massa.

Exportar Snaps

  • Filtros: Por pasta ou tipo de armazenamento (Sync/Local).
  • Formatos: JSON ou CSV.
  • Segurança CSV: Formato ajustado para Excel, evitando fórmulas.

Importar Snaps

  • Opções: Criar novos ou atualizar existentes.
  • Formatos: JSON ou CSV suportados.
  • Recuperação: Botão Restaurar para rollback de segurança.

O que o arquivo tem?O que o SnapIt faz?
ID que já existe na sua baseO Snap é atualizado. Se houver campos vazios no arquivo, os valores atuais do Snap são mantidos.
ID que não existe na baseO Snap é criado como novo (se a opção "Cadastrar faltantes" estiver ativa). Um novo ID será gerado.
Sem ID no arquivoO item é criado como um novo Snap.
Comando (Snap) duplicadoO comando repetido é ignorado para evitar conflitos.
Limites e Requisitos de Segurança por Importação:
  • Máximo de 2000 Snaps ou 25 MB por arquivo.
  • Campos obrigatórios para novos Snaps: Título, Comando (Snap) e Texto.
  • Limites individuais: Título (200), Comando (120) e Texto (50.000 caracteres).
  • Capacidade total máxima combinada: 20.000.000 de caracteres.
Usuários Linux: Caso o popup feche ao selecionar arquivos, use o ícone de monitor no topo do popup.
